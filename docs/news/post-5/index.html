    Version 1.0.1 released!

    27 January 2023

    Yay! We are proud to release version 1.0.1 of the Code Quality Graphing tool, which has the following improvements:

    Fixed the scientific notation for larger numbers

    Added a new synthetic metric Cognitive Complexity per 1000 lines, which has the synthetic name of "CognitiveComplexityPerKLines".

    More information on Cognitive Complexity can be found at https://www.sonarsource.com/docs/CognitiveComplexity.pdf

    Greatly reduced the amount of text output on the command line when the application runs.

    As always, please feel free to ask a question or open an issue in the github repo at https://github.com/larrydiamond/sqgraph
